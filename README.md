# ADIPA Landing Page - Prueba Tecnica Frontend

Landing page de catalogo de cursos inspirada en [adipa.cl/cursos](https://adipa.cl/cursos), implementada en dos stacks tecnologicos.

## Demo en Vivo

| Version | URL |
|---------|-----|
| **Version A** (Next.js) | [https://version-a-sigma.vercel.app](https://version-a-sigma.vercel.app) |
| **Version B** (Laravel) | [https://adipa-landing-project-production.up.railway.app](https://adipa-landing-project-production.up.railway.app) |

## Estructura

```
adipa-landing/
├── shared/          # Datos ficticios compartidos
├── version-a/       # Next.js 16 + Tailwind CSS 4 + TypeScript
└── version-b/       # Laravel 10 + Blade + Stylus + jQuery + Gulp
```
