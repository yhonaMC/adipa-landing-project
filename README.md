# ADIPA Landing Page - Prueba Tecnica Frontend

Landing page de catalogo de cursos inspirada en [adipa.cl/cursos](https://adipa.cl/cursos), implementada en dos stacks tecnologicos.

## Estructura

```
adipa-landing/
├── shared/          # Datos ficticios compartidos
├── version-a/       # Next.js 14 + Tailwind CSS + TypeScript
└── version-b/       # Laravel 9 + Blade + Stylus + jQuery + Gulp
```

## Versiones

| Version | Stack | README |
|---------|-------|--------|
| A | Next.js 14+ (App Router), Tailwind CSS, TypeScript | [version-a/README.md](version-a/README.md) |
| B | Laravel 9+, Blade, Stylus, jQuery, Gulp | [version-b/README.md](version-b/README.md) |

## Datos

Ambas versiones comparten los mismos datos ficticios ubicados en `shared/data/courses.json`. Se incluyen 9 cursos distribuidos en 4 categorias.

## Autor

Desarrollado como prueba tecnica para ADIPA.
